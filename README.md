# 拍图识字微信小程序

该小程序实现以下功能：

* 对印刷体文字进行识别；
* 对手写体文字进行识别；
* 对图片中的表格进行识别，并生成相应的EXCEL文件；
* 图片选择可以通过相册、微信聊天记录以及拍照多种方式；
